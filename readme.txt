Библиотека
Состав группы:
1. Надолько Оля
2. Сегеда Артур
3. Зашаловская Катя
4. Селезнев Вадим

Задача приложения: выступает инструментом для заказа, выдачи и учета книг.

Роли:
Гость
Читатель
Библиотекарь
Администратор

Гость
Просматривает главную страницу
Просматривает список литературы (без информации о наличии книг, оценок, количестве прочтений)
Регистрируется (логин, почта, пароль; опционально: телефон, документ, удостоверяющий личность)
 
Читатель
Просматривает главную страницу
Авторизуется
Если был указан документ, то этот документ подтверждается библиотекарем. Наличие документа увеличивает карму (репутацию; служит для оценки степени надежности и ответственности) пользователя.
Просматривает список литературы (с информацией о наличии книг и оценками, которые формируются пользователями, прочитавшими книгу, количестве прочтений)
В списке для каждой книги отображаются ее характеристики: название, автор, год издания, издательство, жанр, язык, краткое описание  и количество прочтений.
Добавляет понравившиеся книги в закладки.
При определенных уровнях кармы, пользователю становится доступен заказ книг на дом (почтой?) или бронирование (пришел-забрал) книги (не для всех книг, например, редкие книги нельзя выносить из библиотеки). Это действие осуществляется на отдельной страничке, где указываются параметры заказа.
При своевременном возврате книги в библиотеку, пользователю автоматически добавляется немного кармы. При несвоевременном возврате карма убавляется.
После прочтения книги, пользователь может поставить ей оценку.
Может редактировать информацию своего профиля в личном кабинете.
 
Библиотекарь
Просматривает главную страницу
Регистрируется как обычный пользователь, а статус библиотекаря ему выдается админом.
Подтверждает регистрацию пользователя.
Может править учетные данные пользователя по его просьбе.
Принимает заказ от пользователя и выдает ему книгу.
Принимает книгу обратно. Имеет возможность посылать уведомления должникам. Может вмешиваться в процесс зачисления кармы (вдруг кто-то решил подклеить книгу? – карма +5! Или же книга возвращена в плохом состоянии, не вовремя и пр).
Редактирует информацию о книгах и может вносить информацию о новых, списывать старые.
Просматривает список книг, которые находятся у читателей, список пользователей (всех или должников), список заказов (всех или не закрытых).
 
Администратор
Добавляет библиотекарей.
Может редактировать любые учетные записи в системе.
Может все то же, что и библиотекарь.

Виды рейтингов (доступны всем для просмотра):
Книги по оценке.
Книги по кол-ву прочтений.
Пользователи по карме.

Сущности
Книга
Автор
Издательство
Жанр
Полка (физическое местонахождение книги)
Пользователь
Заказ (оформляется каждый раз, когда читатель берет книгу; включает читателя, книгу, дату заказа, ожидаемую дату возврата, дату фактического возврата)
